# DESIGN.md

## 项目与用户画像

**产品名称**：安徽省粮食安全监管分析系统

**目标用户**：省、市、县三级农业执法人员，省级巡查员，系统运维人员

**使用场景**：日常监管工作、现场核查、数据分析、决策支持

## 品牌与视觉方向

**气质与意象**：政务严肃、专业可靠、数据驱动、绿色生态

**视觉关键词**：稳重、清晰、高效、科技感

## Design Tokens

### 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #1A5C9A | 顶部导航、按钮主色、品牌色 |
| 辅色 | #409EFF | 信息提示、次要操作、链接 |
| 警示色 | #E6A23C | 疑似问题、待处理、中等风险 |
| 危险色 | #F56C6C | 确认违规、紧急预警、高风险 |
| 成功色 | #67C23A | 已整改、已结案、低风险 |
| 背景色 | #F5F7FA | 页面背景 |
| 卡片色 | #FFFFFF | 内容卡片 |
| 侧边栏 | 深蓝色渐变 | 左侧导航栏 |

### 字体

- 主字体：PingFang SC, Microsoft YaHei, 系统默认中文字体
- 代码字体：Menlo, Monaco, Consolas
- 字号规范：
  - 标题：20px - 24px
  - 正文：14px
  - 辅助文字：12px

### 圆角

- 卡片圆角：8px (--radius-lg)
- 按钮圆角：6px (--radius-md)
- 标签圆角：4px (--radius-sm)

### 间距

- 页面边距：24px
- 卡片内边距：20px
- 组件间距：16px

### 阴影

- 卡片阴影：`0 2px 12px 0 rgba(0, 0, 0, 0.1)`
- 弹窗阴影：`0 4px 16px 0 rgba(0, 0, 0, 0.15)`

## 布局与响应式

### 整体框架

```
┌─────────────────────────────────────────────────────────────────┐
│  顶部导航栏（60px）                                              │
├────────────┬────────────────────────────────────────────────────┤
│  左侧菜单栏 │           主内容区                                │
│  （220px）  │           (flex-grow)                             │
├────────────┴────────────────────────────────────────────────────┤
│  底部状态栏（32px）                                              │
└─────────────────────────────────────────────────────────────────┘
```

### 响应式断点

- 大屏：≥1440px
- 标准：1280px - 1439px
- 最小支持：1280px

## 组件规范

### 数据卡片

- 背景：白色
- 圆角：8px
- 内边距：20px
- 阴影：轻微投影

### 表格

- 表头背景：#F5F7FA
- 行高：48px
- 边框：1px solid #EBEEF5
- 斑马纹：偶数行 #FAFAFA

### 按钮

- 主按钮：主色背景，白色文字
- 次按钮：白色背景，灰色边框
- 文字按钮：无背景，主色文字

### 状态标签

- 待处理：橙色背景
- 处理中：蓝色背景
- 已完成：绿色背景
- 已驳回：红色背景

### 风险等级

- 高风险：红色 (#F56C6C)
- 中风险：橙色 (#E6A23C)
- 低风险：绿色 (#67C23A)

## 图表配色

- 图表1：主色 #1A5C9A
- 图表2：成功色 #67C23A
- 图表3：警示色 #E6A23C
- 图表4：危险色 #F56C6C
- 图表5：辅色 #409EFF

## 交互与状态

### 状态反馈

- 加载中：骨架屏 + 加载动画
- 空状态：空状态插图 + 引导文字
- 错误状态：错误提示 + 重试按钮

### 动效

- 页面切换：淡入淡出 200ms
- 弹窗：缩放 + 淡入 150ms
- 列表项悬停：背景色变化 150ms

## 设计禁忌

- 禁止使用 AI 味的蓝紫色渐变
- 禁止过度装饰，保持政务系统简洁专业
- 禁止使用过多的圆角和阴影
- 禁止使用花哨的字体和动效
- 数据表格禁止缺少分页和排序功能
